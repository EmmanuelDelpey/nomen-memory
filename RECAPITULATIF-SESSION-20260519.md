**Challenge** : Network latency, partitions, message loss  
**Solution proposée** : CRDT-like + gossip protocols  
**Status** : Connu, solvable, mais nécessite investissement dev

### Garanties cohérence
**Challenge** : CAP theorem, trade-off consistency vs availability  
**Solution proposée** : Strong eventual consistency (acceptable pour cognition)  
**Status** : Paradigme shift requis (accepter SEC vs SC)

**Challenge** : Byzantine faults, prompt injection  
**Solution proposée** : Signature cryptographique flux
