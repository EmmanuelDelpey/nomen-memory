## NOMMAGE = ENCODAGE SPATIAL

**Format** : `[zone][type][coords][hash]`

**Exemple** : `ea35yuee`
- `e` = edges (périphérie)
- `a` = analytical
- `35` = position spatiale
- `yuee` = hash relations

**Le nom encode** :
- Où (zone cortex)
- Quoi (type cognitif)
- Relations (hash)

**Décodable algorithmiquement** sans lire fichier
